### How to use that template

---

1. Clone that template
   > [!NOTE]

<br> 2. Download the dependencies
`bun i or yarn i or npm i`
<br> 3. Initialize new git version

> [!IMPORTANT]

```
rm -rf .git
git init
git add .
git commit -m "Initial commit"
```
